# TrendVue – Social Media Dashboard  

## Live Demo  

[**ifeanyimiracle21.github.io/TrendVue**](https://ifeanyimiracle21.github.io/TrendVue/)

---

## Features  

| Feature | Description |
|---------|-------------|
| **Platform Cards** | Follower count, growth trend, platform icon |
| **Engagement Bar Chart** | Visual comparison of engagement rates (Chart.js) |
| **Animated Counters** | Smooth roll-up on page load |
| **Dark / Light Mode** | Persistent toggle (Sun/Moon) with `localStorage` |
| **Fully Responsive** | Mobile-first grid, adaptive chart height |
| **No Backend** | Pure HTML + CSS + JS – instant preview |

---

## Tech Stack  

- **HTML5** – Semantic structure  
- **CSS3** – Flexbox/Grid, gradients, glassmorphism, dark-mode variables  
- **JavaScript (ES6+)** – DOM manipulation, Chart.js, animations, theme persistence  
- **Chart.js 3.9** – Interactive bar chart  
- **LocalStorage** – Remember theme preference
